# Week 04 Lab Reports

This folder contains lab reports submitted for Week 4 of the Design Thinking course.

## Submitting Your Report

1. Create a folder with your name or student ID: `lab-reports/week-04/your-name/`
2. Copy the template from `templates/LAB_REPORT_TEMPLATE.md`
3. Fill in your report with your work
4. Include any supporting files (images, code, etc.) in your folder
5. Submit a pull request

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for detailed instructions.
